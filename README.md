## Descrição do site

### HTML

inicia com um método de documentação para renderizar HTML – `<!DOCTYPE HTML>` e um método de HTML com uma etiqueta `lang` incerindo a língua portuguesa do Brasil. Entrando na `head`, que é um método que providenciametadados, título (`<title`>), *links* dos ícones, das fontes, do CSS, do JavaScript e o estilo interno (`<style>`), iniciando com título, um metadado de renderização de caracteres e um metadado de autoria.

No início ten-se um alerte construido em JavaScript que exibe a seguinte mensagem: SEJA BEM VINDOS!

no entanto, no corpo, onde há um cabeçalho (`<header>`)que contem um título de primeiro grau (`<h1>`) e as divs (`<div>`), entre os grupos, um conteúdo principal (`<main>`) contendo um formulário (`<form>`). No formulário, temos grupos (`<fieldset>`), dentro dos quais, temos legendas (`<legend`>) dos grupos, rótulos (`<label`>) e caixas de entradas (`<input>`) com um tipo (`type`), um identificador (`id`) e um nome (`name`) pegados pelos rótulos. Os tipos das caixas de entradas podem ser texto, número, telefone, e-mail, data, seleção de opções, seleção de escolha e senha. Também temos uma área de mensagem para escrever (`textarea`). No final, há dois botões (`input`) para enviar e limpar, contendo métodos em JavaScript para clicar as funções em JavaScript.

### JavaScript

É um script interno. Nele, temos uma função que checa se nós quiseremos enviar e abre o diálogo para nos perguntar, senão cancelamos a janela, e outra função que informa que limparemos o formulário ao clicarmos naquele botão.
